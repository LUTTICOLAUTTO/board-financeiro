Atualização e Publicação

  • Substitua os arquivos `.pptx` nesta mesma pasta quando houver versão nova.
  • Faça `git add`, `git commit` e `git push` no repositório.
  • O GitHub Pages publica automaticamente após o push.
  • PDF: tentei automatizar a exportação, mas este ambiente não tem PowerPoint, Keynote nem LibreOffice instalados para converter o `.pptx` com fidelidade.
  • Se você exportar um PDF localmente depois, basta colocá-lo nesta pasta e adicionar um novo botão no `index.html`.
