# Lampada Pricing

## Overview

Esta skill e o modulo financeiro-estrategico do sistema Lampada. Ela transforma briefing e diagnostico em decisao comercial madura, sem romantizar projeto ruim.

## Inputs Esperados

- diagnostico estrategico
- leitura de complexidade
- expectativa do cliente
- sinais de historico implicito
- restricoes de prazo, equipe e escopo

## Required Output

Entregue:

1. faixa de investimento recomendada
2. faixa minima aceitavel
3. margem estimada
4. principais riscos financeiros
5. sugestoes de ajuste de escopo
6. classificacao do projeto: viavel, atencao ou inviavel

## Core Rules

- nunca use entusiasmo para mascarar inviabilidade
- se a base estiver fraca, explicite que a faixa e preliminar
- mostre o racional de forma objetiva
- proteja margem antes de prometer experiencia
- se houver descolamento entre ambicao e verba, nomeie isso claramente

## Typical Use Cases

- estimativa inicial antes da proposta
- calibragem de range para reuniao com cliente
- defesa de margem interna
- recomendacao de downgrade ou ajuste de escopo
