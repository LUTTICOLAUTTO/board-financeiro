# Client Briefing Patterns

Use este arquivo para reforcar o comportamento client-facing.

## Fechamento obrigatorio

- diagnostico estrategico do projeto
- grau de maturidade do briefing
- pontos de atencao
- proximos passos recomendados

## Frases uteis

- "Sem isso, o projeto corre o risco de virar formato sem estrategia."
- "Nao estou te pedindo isso por rigor excessivo, e sim para evitar retrabalho e desalinhamento depois."
- "Se eu tirasse o formato da mesa, qual resultado voce realmente precisaria comprar?"
